# 🌟 FastAPI Greet App

## 📌 Description
A simple FastAPI app with a form-based frontend. Enter your name and get greeted!

## 🧪 Tech Stack
- Backend: Python (FastAPI)
- Frontend: Jinja2 templates
- Deployment: Render

## 🔗 Live Demo
[Visit the app](https://fastapi-greet-app-3.onrender.com)
